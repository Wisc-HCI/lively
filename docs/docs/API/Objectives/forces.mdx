# Forces

Forces Objectives proximate the physcial forces applied to the robot.

| Objective | Goals | Description |
| --- | --- | --- |
| Gravity | none | Add a downward force to the position of a link |

## Parameters for constructing Forces Objectives
|Parameter|Type|Optional|Defauly value/behavior|Description|
|---|---|---|---|---|
|`name`|string|no|none|Name can be arbitrary and should be unqiue|
|`weight`|float|no|none|Indicates the prioritization of the objective|
|`link`|string|no|none|The name of the link this objective is applied to|
