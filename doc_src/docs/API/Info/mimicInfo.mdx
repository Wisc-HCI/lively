# Mimic Info

`MimicInfo` is an optional subset of [`JointInfo`](jointInfo) is present when the given joint mimics another. Contains information about the identity of the other joint, as well as multiplier scaling and offset values.

| Parameter | Type | Description |
|---|---|---|
|`joint`|string|The corresponding [`joint`](jointInfo)|
|`multiplier`|float|A value used for scaling|
|`offset`|float|A value used as an offset|
