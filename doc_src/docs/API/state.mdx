# State
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

The `State` object is the response back after calling [`solve`](Solver/Methods/solve). 
It contains the state of the robot in terms of joint and frames, as well as some diagnostic information regarding [`proximity`](../API/Info/proximityInfo) of various [`shapes`](../API/Shapes/index) and the `center-of-mass` of the robot. 


| Parameter |Type| Description |
|---|---|--|
|`origin`|isometry consisted of translation and rotation|The transform of the root of the robot|
|`joints`|lookup table(string, float)|A lookup table of the joint values for each movable joint|
|`frames`|lookup table(string, [`TransformInfo`](../API/Info/transformInfo))|A lookup table of the positions/orientations for each link. Can either be `world` or `robot frame`|
|`proximity`|list of [`ProximityInfo`](../API/Info/proximityInfo)|A list of all shapes that are currently tracked and that reach close enough proximity to potentially factor into collision detection|
|`center_of_mass`|list of float|A translation (vector) indicating the current center of mass of the robot|
